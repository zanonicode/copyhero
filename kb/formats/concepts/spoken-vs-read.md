# Falado vs Lido — Regras de Cadência

> **Propósito**: Regras de escrita para distinguir copy que será ouvida (VSL, Reels) de copy que será lida (sales page, blog).
> **Quando usar**: Ao escrever VSL, roteiro de Reels ou qualquer formato com áudio/apresentação oral.

## O que é

O maior erro em VSL e Reels no mercado BR: escrever o roteiro como se fosse uma sales page. O resultado é uma leitura artificial — o criador fala como se estivesse recitando, o áudio perde ritmo, o espectador desconecta.

Texto falado e texto lido têm cadências fundamentalmente diferentes. Entender essa diferença é o que separa um VSL que prende de um que parece apresentação escolar.

## As Diferenças Fundamentais

| Dimensão | Texto Lido (Sales Page / Blog) | Texto Falado (VSL / Reels) |
|----------|-------------------------------|---------------------------|
| **Comprimento das frases** | Frases médias (15-25 palavras) toleradas | Frases curtas (5-12 palavras); respiração natural |
| **Parágrafos** | 2-4 linhas aceitáveis | Não existem parágrafos — cada frase é uma linha |
| **Vocabulário** | Pode ser um pouco mais formal | Conversacional — como se falasse com um amigo |
| **Repetição** | Evitar repetição — parece descuido | Repetição deliberada — reforça pontos, cria ritmo |
| **Pontuação** | Gramatical | Reticências (...) e travessões (—) marcam pausa |
| **Marcadores visuais** | Bullets, H2, negrito | Não existe — a voz é o único marcador |
| **Transições** | "Como resultado", "Portanto" | "Olha.", "E sabe o que acontece?", "Mas aqui vai a virada." |

## Regras de Escrita para Texto Falado

### Regra 1: Uma frase por linha no roteiro
Não escreva em parágrafos. Cada linha é uma resposta à pergunta implícita: "você vai continuar prestando atenção?"

```
ERRADO:
"O maior problema de quem tenta crescer no Instagram é que passa horas criando conteúdo de qualidade mas não entende como o algoritmo funciona e acaba vendo páginas com menos esforço crescendo mais rápido."

CERTO:
"Você posta todo dia.
Com qualidade.
Com consistência.
E o algoritmo enterra.
Por quê?"
```

### Regra 2: Leia em voz alta antes de finalizar
Se você trava em alguma frase ao ler em voz alta, o apresentador também vai travar. Reescreva até que flua naturalmente.

### Regra 3: Reticências e travessões são seus aliados
Eles marcam pausas deliberadas — onde o orador respira, onde o espectador processa.

```
"Tinha chegado no fundo...
...e foi exatamente lá que encontrei a saída."
```

### Regra 4: Repita pontos-chave com variação
Em texto lido, repetição parece preguiça. Em texto falado, repetição cria ênfase e ajuda memória.

```
"Não é o produto que vende.
Não é o tráfego.
Não é nem a oferta.
É a mensagem."
```

### Regra 5: Transições conversacionais BR
Evite conectores formais em texto falado. Use transições que soam como conversa:

| Formal (evitar em VSL) | Conversacional (usar em VSL) |
|------------------------|------------------------------|
| "Portanto" | "Então." / "Aí." |
| "No entanto" | "Só que." / "Mas aí." |
| "Como resultado" | "E sabe o que acontece?" |
| "Em conclusão" | "Olha." / "Fica ligado nisso." |

## Velocidade e Ritmo (VSL)

- **Velocidade média BR para VSL**: 130 palavras por minuto
- **VSL de 20 minutos**: ≈ 2.600 palavras de roteiro
- **Hook dos primeiros 30 segundos**: ≈ 65 palavras — devem ser as mais fortes do roteiro

## Checklist de Roteiro Falado

```
[ ] Uma frase por linha no roteiro
[ ] Nenhuma frase acima de 15 palavras consecutivas
[ ] Leu em voz alta sem travar
[ ] Transições são conversacionais, não formais
[ ] Hook dos primeiros 30s é a promessa mais forte
[ ] Reticências marcam pausas dramáticas
```
