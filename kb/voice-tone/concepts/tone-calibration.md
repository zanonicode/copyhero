# Calibração de Tom

> **Propósito**: Regras para ajustar o espectro formal↔informal dentro do arquétipo escolhido, por avatar e formato.
> **Quando usar**: Ao escrever qualquer copy — o arquétipo define a voz, a calibração define a intensidade.

## O que é

O arquétipo define o *personagem* (mentor próximo, autoridade técnica, etc.). A calibração define a *intensidade* — o quão formal ou informal, o quão direto ou suave, o quão técnico ou acessível.

Dois copywriters usando o arquétipo "mentor próximo" podem produzir copy completamente diferente se um calibrar para um designer de 28 anos em São Paulo e o outro para um contador de 45 anos em Porto Alegre. A calibração é o ajuste fino que faz a copy soar feita *para essa pessoa*.

## O Espectro BR

```
Muito informal ←────────────────────────────────→ Muito formal
     |              |              |              |
  "Cê sabe"    "Você sabe"   "Caro leitor"   "Conforme"
  Regionalismos  Neutro BR    Cortês formal    Técnico
  Gírias         sem gíria    sem jargão      com jargão
```

**BR-neutro** é o ponto de partida para a maioria dos avatares. Ajusta em uma direção ou outra baseado nos sinais do briefing.

## Fatores de Calibração

### Fator 1: Faixa Etária do Avatar

| Faixa | Tendência | Exemplo de ajuste |
|-------|-----------|------------------|
| 18-28 anos | Informal-médio | "Bora" / "cê" OK se o arquétipo permite |
| 29-40 anos | Neutro BR | "Você" padrão; sem gírias marcadas |
| 41+ anos | Neutro a formal-leve | "Você" padrão; evitar gírias; respeito implícito |

**Cuidado**: faixa etária é sinal, não regra. Designer de 45 anos no Instagram pode preferir tom informal; advogado de 30 anos pode querer formal.

### Fator 2: Nicho e Contexto Profissional

| Contexto | Calibração |
|----------|-----------|
| Saúde / terapia | Neutro-cálido — sem agressividade, sem pressão |
| Negócios / marketing | Neutro a informal — energia é bem-vinda |
| Direito / finanças | Neutro a formal — precisão é sinal de credibilidade |
| Fitness / performance | Informal-energético — o nicho aceita intensidade |
| Criação de conteúdo | Informal-médio — a autenticidade é o produto |

### Fator 3: Plataforma

| Plataforma | Calibração recomendada |
|------------|----------------------|
| Reels / TikTok | Informal — câmera = conversa pessoal |
| Post IG | Informal-médio |
| Email | Neutro — mais pessoal que blog, menos que DM |
| Blog / SEO | Neutro a formal-leve — ranqueamento pede clareza |
| Sales page | Arquétipo puro — mais controlado que Reels |

### Fator 4: Preço do Produto

| Faixa de Preço | Calibração |
|----------------|-----------|
| Gratuito / baixo (< R$ 100) | Informal está OK — decisão de baixo risco |
| Médio (R$ 100-1.000) | Neutro — equilíbrio entre conexão e credibilidade |
| Alto (R$ 1.000+) | Neutro a formal-leve — a credibilidade pesa mais |

## Como Calibrar na Prática

### Teste de leitura em voz alta

Leia o primeiro parágrafo em voz alta como se fosse para o avatar. Se soar como uma conversa natural, está calibrado. Se soar como discurso ou como mensagem de texto demais, ajuste.

### Substituições de calibração

**Mover para mais informal:**
- "Você" → "Cê" (apenas se arquétipo permite)
- "Portanto" → "Então"
- "Resultado" → "Resultado mesmo"
- Parágrafos mais curtos, frases mais curtas

**Mover para mais formal:**
- "Cê" → "Você"
- Remover reticências excessivas
- Frases com estrutura mais completa (sujeito + verbo + objeto)
- Vocabulário técnico do nicho com explicação

## Manutenção de Tom ao Longo de Copy Longa

O maior risco em copy de 3.000+ palavras é o escorregamento de tom — começa informal e vai ficando formal na seção de prova social, ou começa formal e fica coloquial demais no CTA.

**Protocolo de revisão de tom**:
1. Leia apenas os primeiros 2 parágrafos
2. Leia o meio da copy (seção de prova social)
3. Leia o CTA
4. Os 3 devem soar como a mesma pessoa falando

Se não soarem, identifique onde escorregou e normalize.
